Name Description Type Example
All Determines whether all elements of a sequence satisfy a condition. Iterative Will check to see if all ages are greater than 0
var result = personList.All(x => x.Age > 0);
Any Determines whether any element of a sequence exists or satisfies a condition. Iterative Will check to see if any ages are greater than 0
var result = personList.Any(x => x.Age > 0);
Average Computes the average of a sequence of numeric values. Iterative Computes the average of all ages.
var result = personList.Average(x => x.Age);
Count Returns the number of elements in a sequence. Iterative var result = personList.Count();
Distinct Returns distinct elements from a sequence. Will not display any duplicates. Deferred var result = personList.Select(x => x.Location).Distinct();
foreach(var person in result) Console.WriteLine(person);
ElementAt Returns the element at a specified index in a sequence. Deferred var result = personList.ElementAt(21);
First Returns the first element of a sequence. Iterative var result = personList.First();
FirstOrDefault Returns the first element of a sequence, or a default value if no element is found. Iterative var result = personList.FirstOrDefault(x => x.Age > 0);
Last Returns the last element of a sequence. Iterative var result = personList.Last();
LastOrDefault Returns the last element of a sequence, or a default value if no element is found. Iterative var result = personList.LastOrDefault(x => x.Age < 0);
Max Returns the maximum value in a sequence of values. Iterative var result = personList.Max(x => x.Age);
Min Returns the minimum value in a sequence of values. Iterative var result = personList.Min(x => x.Age);
OfType Filters the elements of an IEnumerable based on a specified type (string, int, Person, etc...) Deferred "Running 'OfType<string>' on the object will only grab '3' and 'Some other string'";
var array = new object[] {1, "3", "Some other string", 10};
var afterOfType = array.OfType<string>();
foreach (var item in afterOfType) Console.Write($"{item}");
OrderBy Sorts the elements of a sequence in ascending order. Deferred Sorts the list by Last Name in ascending order.
var persons = personList.OrderBy(x => x.LastName);
foreach (var person in persons) Console.WriteLine(person.WhateverProperties);
OrderByDescending Sorts the elements of a sequence in descending order. Deferred Sorts the list by Last Name in descending order.
var persons = personList.OrderByDescending(x => x.LastName);
foreach (var person in persons) Console.WriteLine(person.WhateverProperties);
Reverse Inverts the order of the elements in a sequence. Deferred var result = personList.Reverse();
foreach (var person in result) Console.WriteLine(person.WhateverProperties);
Select Projects each element of a sequence into a new form. Deferred Creates a new list of your selected elements. In this case a new list of LastName, FirstName
var result = personList.Select(x => $"{x.LastName}, {x.FirstName}");
foreach (var person in result) Console.WriteLine(person);
Skip Bypasses a specified number of elements in a sequence and then returns the remaining elements. (Skip, SkipLast and SkipWhile) Deferred Skips displaying element 42 and any element prior.
var result = personList.Skip(42);
foreach (var person in result) Console.WriteLine(person.WhateverProperties);

Skips displaying the last 5 elements.
var result = personList.SkipLast(5);
foreach (var person in result) Console.WriteLine(person.WhateverProperties);

Skips displaying elements with an age of 25. When an element matches the criteria, SkipWhile stops and the rest of the list is displayed.
var result = personList.SkipWhile(x => x.Age <= 25);
foreach (var person in result) Console.WriteLine(person.WhateverProperties);
Sum Computes the sum of a sequence of numeric values. Iterative Computes the sum of all the ages.
var summedAges = personList.Sum(x => x.Age);
Take Returns a specified number of contiguous elements from the start of a sequence. (Take, TakeLast and TakeWhile) Deferred Takes and displays only the first 10 elements
var result = personList.Take(10);
foreach (var person in result) Console.WriteLine(person.WhateverProperties);

Takes and displays only the last 5 elements.
var result = personList.TakeLast(5);
foreach (var person in result) Console.WriteLine(person.WhateverProperties);

Takes and displays all elements until the condition is met.
var result = personList.TakeWhile(x => x.Age <= 25);
foreach (var person in result) Console.WriteLine(person.WhateverProperties);
ToDictionary Creates a Dictionary<TKey,TValue> from an IEnumerable<T>. Deferred Creates a dictionary with a key (Id) value (LastName) pair.
var result = personList.ToDictionary(x => x.Id, x => x.LastName);
foreach (var (key, value) in result) Console.WriteLine($"{key}: {value}");
Where Filters a sequence of values based on a predicate/criteria. Deferred Filter ages greater than 50.
var afterWhere = personList.Where(x => x.Age > 50);
foreach (var person in afterWhere) Console.WriteLine(person.WhateverProperties);

Filter ages greater than 50 and where Id is less than 50.
var afterWhere = personList.Where(x => x.Age > 50 && x.Id < 50);
foreach (var person in afterWhere) Console.WriteLine(person.WhateverProperties);
